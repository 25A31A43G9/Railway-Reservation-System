# 🚆 Railway Reservation System

A simple and interactive **Railway Reservation System** developed in **C++** using Object-Oriented Programming (OOP) concepts. This project simulates basic railway ticket reservation operations such as booking tickets, cancelling tickets, and checking seat availability through a menu-driven interface.

## 📌 Features

✅ Book train tickets

✅ Cancel booked tickets

✅ Check available seats

✅ Real-time seat count updates

✅ User-friendly menu-driven system

✅ Object-Oriented Programming implementation

## 🛠️ Technologies Used

* C++
* Object-Oriented Programming (OOP)
* Functions and Classes
* Conditional Statements
* Loops and Switch Cases

## 🚂 System Overview

The system starts with **50 available seats**. Users can:

* 🎫 Book tickets if seats are available
* ❌ Cancel previously booked tickets
* 📊 Check the current seat availability
* 🚪 Exit the application

The seat count is automatically updated after every booking or cancellation.

## 📋 Menu Options

1️⃣ Book Ticket

2️⃣ Cancel Ticket

3️⃣ Check Seat Availability

4️⃣ Exit

## 🔄 Working

When a user selects the booking option, the system asks for the number of seats required and verifies whether enough seats are available. If seats are available, the booking is confirmed and the available seat count is reduced accordingly. The cancellation feature increases the available seat count when seats are cancelled. Users can also check the current number of available seats at any time through the availability option.


4. Run the executable:

